## Problem Highlight: Roman to Integer (#13)

**Description:**

Converts a Roman numeral string to an integer using Java.
Implements a simple traversal with conditional subtraction logic.

### Features 
- Uses 'HashMap<Character, Integer>' for constant-time lookups
- Handles subtractive notation (e.g., IV, IX, XL, XC, CD, CM)
- Efficient single-pass O(n) algorithm

### What I learned
- Reinforced understanding of string traversal and index handling
- Learned to use **HashMap** effectively for character-to-integer mapping
- Practiced algorithmic problem-solving and runtime analysis

### Technologies
- **Java**
- **HashMap, String Traversal**, and basic algorithmic logic.

## Problem Highlight: Two Sum (#1)

**Description:**

Given an array of values and a target value, find the indices of the numbers that add up to the target.

### Features
- Efficient O(n) time complexity using HashMap
- Single-pass solution with constant time lookup
- Returns indices in any order

### What I learned
- How to optimize from O(n^2) brute force to O(n) using hash maps.
- The importance of choosing the right data structure.
- How to think about complement patterns (i.e., target - current = needed)

### Technologies
- Java
- HashMap data structure
- Array manipulation
